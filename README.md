# covid19-data-analysis-and-prediction
COVID-19 data analysis and prediction using Python, NumPy, Pandas, Matplotlib, Seaborn, and Scikit-learn (Linear Regression).
# 🚀 COVID-19 Data Analysis & Prediction Project

## 📌 Overview

This project focuses on analyzing COVID-19 data to extract meaningful insights and predict future trends using Machine Learning.

The goal of this project is to understand how data can be transformed into actionable insights and support decision-making.

---

## 📊 Features

* Data Cleaning and Preprocessing
* State-wise Analysis of COVID-19 Cases
* Identification of Top Affected States
* Death Rate Calculation
* Risk Level Classification (High, Medium, Low)
* Future Case Prediction using Linear Regression

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn

---

## 📂 Dataset

The dataset contains 10,000+ records including:

* Confirmed Cases
* Recovered Cases
* Deaths
* State-wise Data

---

## 📈 Key Insights

* Identified the most affected states based on confirmed cases
* Calculated average death rate across regions
* Classified regions into different risk categories
* Observed trends in case growth over time

---

## 🤖 Machine Learning Model

A **Linear Regression model** was used to:

* Analyze trends in COVID-19 cases
* Predict future cases for upcoming days
